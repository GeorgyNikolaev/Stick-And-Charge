Чтобы начать, нужно просто запустить файл "main.py"
Далее откроется окно с параметрами, для удобства нужно сместить его вправо, так как окно с симуляцией будет открываться в левой части экрана. 
Управление интуитивно понятное. Просто нужно ввести любые значение в поле ввода и нажать кнопку "Симуляция". Под кнопкой показан модуль силы.
В файлах с кодом я оставил немного комментариев, для понимания. Возможно суть кода будет не понятна, поэтому я готов его защитить.
Возможно придется установить пару библиотек, если они у вас отсутствуют.
